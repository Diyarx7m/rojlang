MÎNAK — EXAMPLES

Koda RojLang di kar de

Real programs written entirely in Kurdish keywords.

📦 Guhêrbar — Variables
# Danasîna guhêrbar
nav nav_min = "Roj"
nav temen = 25
nav xwendekar = rast

nivîs("Nav:", nav_min)
nivîs("Temen:", temen)
🔀 Şert — Conditionals
nav x = 10

eger x > 5:
    nivîs("Mezin ji 5")
yan eger x == 5:
    nivîs("Wekhev 5")
yan:
    nivîs("Biçûk ji 5")
🔁 Xulama dem — While Loop
nav i = 1

dem i <= 5:
    nivîs("Hejmar:", i)
    i += 1
🔄 Xulama bo — For Loop
nav reng = ["sor", "kesk", "şîn"]

bo r di reng:
    nivîs("Reng:", r)
⚙️ Fonksîyon — Function
fonksîyon silav(ism):
    vegere "Silav, " + ism + "!"

nivîs(silav("Roj"))
# → Silav, Roj!
🎨 Hunerê Kurdî — Graphics
# Kurdish art window
nav xalan = pencere("Hunerê Kurdî", "500x400")
nav kevça = kevir(xalan, 500, 400, "#0a0a0f")

çargoşe(kevça, 50, 50, 200, 200, "#e63946")
xember(kevça, 260, 50, 450, 240, "#f0a500")
xêz(kevça, 50, 300, 450, 300, "#06d6a0", 3)
nivîskar(kevça, 250, 360, "☀ RojLang", "#f0a500", 18)

nîşan(xalan)
🎮 Lîstika Texmînkirinê
nav veşartî = hilbijêre(1, 10)
nav çalak = rast

dem çalak:
    nav texmîn = int(xwestin("Texmîn: "))
    eger texmîn == veşartî:
        nivîs("🎉 Rast!")
        nav çalak = çewt
SAZKIRIN — INSTALLATION

Li ser Windows

Carekê saz bike — li her cihî bikar bîne. Install once — use anywhere.

1

⬇ Hemûyan daxistin — Download All

Bişkoja "Download All" ya jorîn bitikîne — 3 pel tên daxistin.
Click "Download All" above — 3 files will download.

2

📁 Hemûyan di heman peldankê de bixin

Hemû pelên daxistî di heman peldankê de bixin (mînak: Desktop).
Put all files in the same folder (e.g. Desktop).

3

⚙️ Mişkê rast → Run as administrator

Li ser install_rojlang.bat mişkê rast bitikîne → "Run as administrator".
This is important — it needs admin rights to set up PATH.

4

🖥️ Terminalek nû veke — Open new terminal

Piştî sazkirinê CMD an Terminal nû veke (PATH tenê di terminalên nû de dixebite).
After install, open a new terminal window.

5

🚀 Bikar bîne!

Ji her cihî binivîse rojlang filename.ku — an du-klîk li ser her .ku pelê bike! ✅

CMD — anywhere on your PC
# Ji her peldankê — From any folder:
rojlang game.ku
rojlang test.ku

# Debug mode:
rojlang game.ku --debug

# Bi rêça temam — with full path:
rojlang "C:\Users\te\Desktop\test.ku"
PIRSGIRÊK — TROUBLESHOOTING

Çareseriya Pirsgirêkan

Common problems and how to fix them — Pirsgirêkên gelemperî û çareyên wan.

🐍
'python' is not recognized — Python nehat nasîn
▼
PIRSGIRÊKA HERÎ GELEMPERÎ

Ev tê wê wateyê ku Python li ser komputera te saz nebûye. RojLang bi Python dixebite — divê pêşî Python saz bikî.

How to install Python
1. Biçe: https://python.org/downloads
2. "Download Python 3.x" bitikîne
3. Sazkerê veke
4. ☑ "Add Python to PATH" ← Pir girîng! Never forget this!
5. "Install Now" bitikîne
6. CMD-ê nû veke û test bike:
   python --version
Piştî Python sazkirinê → install_rojlang.bat ji nû ve bixebitîne ✓
⚠️
Python saz kir lê hîn jî 'python' nayê nasîn
▼
ÇARESERÎ HÊSAN

Dibe ku dema sazkirinê "Add Python to PATH" çeknekir. Sazkerê Python dîsa veke → "Modify" → "Add to PATH" têxe.

An jî bi destan: Search → "Environment Variables" → Path → Edit → New → rêça Python lê zêde bike.
Mînak: C:\Users\te\AppData\Local\Programs\Python\Python312

🔧
'rojlang' nayê nasîn piştî sazkirinê — rojlang not found
▼
ÇARESERÎ HÊSAN

Terminalek nû veke — PATH tenê di terminalên nû de dixebite. Terminalê kevin nagihîje PATH-ê nû.

Heke hîn nexebite, check bike: C:\RojLang\rojlang.bat heye? Heke nebe, install_rojlang.bat ji nû ve weke admin bixebitîne.

🛡️
Installer xelet qediya — Installer failed or did nothing
▼
SEDEM GELEMPERÎ

Installer bêyî "Run as administrator" hat xebitandin. Registry û PATH mafên rêvebir hewce dike.

Çareserî: Mişkê rast → "Run as administrator" → Enter bike. Tenê carekê dîsan.

Carekê dîsa weke admin bixebitîne — that's it ✓
📄
Du-klîk li .ku file naxebite — Double-click opens wrong app
▼
ÇARESERÎ

Bi destan file association rast bike — CMD weke admin veke û ev bixebitîne:

CMD (Admin)
reg add "HKCR\.ku" /ve /d "RojLangFile" /f
reg add "HKCR\RojLangFile\shell\open\command" /ve /d "cmd.exe /k python \"C:\RojLang\interpreter.py\" \"%%1\"" /f
🔤
UnicodeDecodeError — Tîpên Kurdî xelet xuya dikin
▼
PIRSGIRÊKA KODKIRINÊ

Pelê .ku divê bi UTF-8 were hilanîn. Edîtorê xwe check bike:

Fix by editor
# VS Code:
# Guşeya jêrîn rastê → UTF-8 divê be

# Notepad++:

# Save As → Encoding → UTF-8
